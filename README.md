# CampusConnect - Event Management System (C++)

## 📌 Description
A console-based application to manage college events efficiently and solve communication gaps in event announcements.

## 🚀 Features
- Add / Delete / Search Events
- Student Registration
- Notification System (Queue)
- Participant Tracking
- File Persistence

## 🧠 Data Structures Used
- unordered_map → Event storage
- queue → Notification handling (FIFO)
- vector → Participants list

## 💡 Future Improvements
- Web/App interface
- Database integration
- Autocomplete search (Trie)
